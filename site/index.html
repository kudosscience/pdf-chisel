PDF Chisel

Edit text and replace images directly in your PDFs. 100% offline, open source, and free.

100% Offline Open Source Windows / macOS / Linux
Download for Windows All Platforms

Features

Everything you need to edit PDFs, nothing you don't.

✎

Edit Text In Place

Double-click any text on the PDF and type your changes. What you see is what you get.

📷

Replace Images

Swap embedded images with your own PNG or JPEG files. Click the image and choose a replacement.

↺

Undo & Redo

Full edit history. Ctrl+Z to undo, Ctrl+Shift+Z to redo. Never worry about mistakes.

📄

Multi-Page

Navigate, zoom, and edit any page. Thumbnail sidebar for quick page previews.

🔒

100% Offline

Your files never leave your computer. No cloud, no accounts, no subscriptions.

🖥

Cross-Platform

Runs natively on Windows, macOS (Intel & Apple Silicon), and Linux.

Download

Pick the installer for your operating system.

Platform Download Notes
Windows PDF.Chisel.Setup.0.2.0.exe Run the installer and follow the prompts
macOS (Apple Silicon) PDF.Chisel-0.2.0-arm64.dmg Open the .dmg, drag to Applications
macOS (Intel) PDF.Chisel-0.2.0.dmg Open the .dmg, drag to Applications
Linux PDF.Chisel-0.2.0.AppImage chmod +x *.AppImage then run
Not sure which Mac you have? Click the Apple menu  → About This Mac. If the chip says "Apple M1" / "M2" / "M3", choose Apple Silicon. Otherwise choose Intel.
